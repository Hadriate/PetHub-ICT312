About the project

PetHub brings a small pet store online with a cleaner customer experience.

PetHub is designed as a user-friendly e-commerce information system that helps customers browse and order pet products while giving store owners an easier way to organize products, orders, and stock.

🎯

Purpose

Make it easier for customers to find food, toys, grooming products, and accessories in one place.

🔐

Security focus

The final build can use secure authentication, role-based access, input validation, and protected admin pages.

⚙️

Technology

HTML, CSS, and JavaScript are used in this stage. PHP and MySQL are planned for full functionality.

Scope

Customer and admin features

The customer side includes product browsing, account pages, cart, and checkout. The admin side is planned to support product updates, inventory monitoring, and order management.

  • Home, About, Products, Product Details
  • Login, Register, Cart, Checkout, Contact
  • Admin dashboard with sales and stock summary
